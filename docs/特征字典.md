### 1. 时域特征 (Time Domain Features)

时域特征直接基于信号波形的统计特性，反映运动的强度、平稳性和分布形态。

| 特征名称 | 物理含义 | 计算逻辑/说明 |
| :--- | :--- | :--- |
| **mean_x/y/z** | **基线偏移/姿态指示** | 信号在 X/Y/Z 轴上的算术平均值。若数据未去重力，反映传感器倾斜角度；若已去重力，反映传感器零点漂移。当前数据均值接近0，疑似已去重力或处于动态平衡。 |
| **std_mag** | **运动剧烈程度** | 合加速度（Magnitude）的标准差。反映信号偏离均值的程度，值越大代表运动越剧烈、波动越大。 |
| **rms_mag** | **信号能量** | 合加速度的均方根。反映信号包含的能量大小，是衡量运动强度的核心指标。 |
| **ptp_mag** | **振幅范围** | 合加速度的峰峰值 (Max - Min)。反映窗口内信号变化的最大幅度。 |
| **skewness_mag** | **波形对称性** | 合加速度的偏度。反映数据分布的不对称性。接近 0 表示分布对称（如匀速行走），显著非零表示存在单向冲击或非线性运动特征。 |
| **kurtosis_mag** | **信号冲击性** | 合加速度的峰度。反映信号分布的尾部厚度。高值通常对应跑步、跳跃等冲击性动作。需注意计算的是标准峰度还是超额峰度。 |
| **zcr_mag** | **归一化过零率** | 合加速度穿过均值线的次数与窗口长度的比率。用于粗略估计信号频率，对噪声敏感。 |
| **sma** | **总运动量** | 信号幅值面积。三轴加速度绝对值之和的平均，反映整体运动负荷。 |

### 2. 频域特征 (Frequency Domain Features)

频域特征通过 FFT 或 PSD（功率谱密度）变换获得，反映运动的节奏、规律性和频率分布。

| 特征名称 | 物理含义 | 计算逻辑/说明 |
| :--- | :--- | :--- |
| **dom_freq** | **主频率 (步频)** | 功率谱密度 (PSD) 中能量最大点对应的频率。通常对应人的步频或动作重复频率 (Hz)。 |
| **spectral_centroid** | **频谱重心** | 频谱的“质心”位置。反映频谱能量集中的位置，值越大说明高频分量越多（动作越快/越剧烈）。 |
| **spectral_entropy** | **频谱复杂度** | 基于归一化功率谱密度计算的香农熵。反映信号的混乱程度。当前数据中该特征区分度较低，建议检查计算时是否排除了直流分量。 |
| **low_freq_ratio** | **低频能量占比** | 0-3Hz 频段的能量占总能量的比例。人体日常活动（走、跑、坐）的主要能量通常集中在 0-5Hz 低频段。 |

### 3. 特征与动作的关联性分析 (基于当前数据)

根据生成的 `feature_matrix.csv`，不同特征对动作类别的区分能力如下：

1.  **区分“静止”与“运动”**：
    *   主要依赖 **`rms_mag`** 和 **`std_mag`**。
    *   *现象*：静止类（如 Label 2）的能量值极低（~0.006），而运动类（如 Label 0/5）能量值较高（~0.14）。

2.  **区分“运动类型” (如走路 vs 跑步)**：
    *   主要依赖 **`dom_freq`** (主频) 和 **`spectral_centroid`**。
    *   *现象*：Label 5 的主频较低 (~2.9Hz)，Label 0 的主频较高 (~5.0Hz)，说明两者节奏不同。
