\# 📊 HAR 特征字典 (Feature Dictionary)



\*\*项目阶段\*\*：D4 特征工程

\*\*数据来源\*\*：`windowed\_dataset.csv` (加速度计数据)

\*\*特征总数\*\*：14 (不含 ID 和 Label)



\---



\## 1. 时域特征 (Time Domain Features)



时域特征直接基于信号波形的统计特性，反映运动的强度、平稳性和分布形态。



| 特征名称 | 物理含义 | 计算逻辑/说明 |

| :--- | :--- | :--- |

| \*\*mean\_x/y/z\*\* | \*\*直流分量/重力分量\*\* | 信号在 X/Y/Z 轴上的算术平均值。反映传感器相对于重力方向的倾斜角度或静态偏移。 |

| \*\*std\_mag\*\* | \*\*运动剧烈程度\*\* | 合加速度（Magnitude）的标准差。反映信号偏离均值的程度，值越大代表运动越剧烈、波动越大。 |

| \*\*rms\_mag\*\* | \*\*信号能量\*\* | 合加速度的均方根。反映信号包含的能量大小，是衡量运动强度的核心指标。 |

| \*\*ptp\_mag\*\* | \*\*振幅范围\*\* | 合加速度的峰峰值 (Max - Min)。反映窗口内信号变化的最大幅度。 |

| \*\*skewness\_mag\*\* | \*\*波形对称性\*\* | 合加速度的偏度。反映数据分布的不对称性。正值表示波峰在左侧（下楼梯常见），负值表示波峰在右侧（上楼梯常见）。 |

| \*\*kurtosis\_mag\*\* | \*\*冲击感/平坦度\*\* | 合加速度的峰度。反映信号中冲击成分的比例。跑步或跳跃等冲击性动作通常具有较高的峰度。 |

| \*\*zcr\_mag\*\* | \*\*频率粗略估计\*\* | 合加速度的过零率。信号穿过均值线的频率，用于粗略估计信号的主频，区分静止与运动。 |

| \*\*sma\*\* | \*\*总运动量\*\* | 信号幅值面积。三轴加速度绝对值之和的平均，反映整体运动负荷。 |



\## 2. 频域特征 (Frequency Domain Features)



频域特征通过 FFT 或 PSD（功率谱密度）变换获得，反映运动的节奏、规律性和频率分布。



| 特征名称 | 物理含义 | 计算逻辑/说明 |

| :--- | :--- | :--- |

| \*\*dom\_freq\*\* | \*\*主频率 (步频)\*\* | 功率谱密度 (PSD) 中能量最大点对应的频率。通常对应人的步频或动作重复频率 (Hz)。 |

| \*\*spectral\_centroid\*\* | \*\*频谱重心\*\* | 频谱的“质心”位置。反映频谱能量集中的位置，值越大说明高频分量越多（动作越快/越剧烈）。 |

| \*\*spectral\_entropy\*\* | \*\*规律性/复杂度\*\* | 频谱的香农熵。反映信号的混乱程度。\*\*周期性运动\*\*（如走路）熵值较低，\*\*随机噪声\*\*或复杂动作熵值较高。 |

| \*\*low\_freq\_ratio\*\* | \*\*低频能量占比\*\* | 0-3Hz 频段的能量占总能量的比例。人体日常活动（走、跑、坐）的主要能量通常集中在 0-5Hz 低频段。 |

\## 3. 特征与动作的关联性分析 (基于当前数据)



根据生成的 `feature\_matrix.csv`，不同特征对动作类别的区分能力如下：



1\.  \*\*区分“静止”与“运动”\*\*：

&#x20;   \*   主要依赖 \*\*`rms\_mag`\*\* 和 \*\*`std\_mag`\*\*。

&#x20;   \*   \*现象\*：静止类（如 Label 2）的能量值极低（\~0.006），而运动类（如 Label 0/5）能量值较高（\~0.14）。



2\.  \*\*区分“运动类型” (如走路 vs 跑步)\*\*：

&#x20;   \*   主要依赖 \*\*`dom\_freq`\*\* (主频) 和 \*\*`spectral\_centroid`\*\*。

&#x20;   \*   \*现象\*：Label 5 的主频较低 (\~2.9Hz)，Label 0 的主频较高 (\~5.0Hz)，说明两者节奏不同。



3\.  \*\*区分“动作规律性”\*\*：

&#x20;   \*   主要依赖 \*\*`spectral\_entropy`\*\*。

&#x20;   \*   \*现象\*：Label 2 (静止) 的熵值很高 (\~6.3)，说明主要是白噪声；而 Label 0/5 的熵值较低 (\~2.2)，说明具有明显的周期性规律。
