# HAR 数据采集协议 v1.0

> **文档版本**：v1.0
> **适用场景**：基于 ESP32-S3 + MPU6050 的人体活动识别 (HAR)
> **核心原则**：**被试独立 (Subject-Independent)** & **数据纯净度优先**

---

## 1. 实验目的

采集高质量、标注清晰的 IMU 数据，用于训练能够区分 **8 种人体活动**的分类器。
本次采集重点解决两个核心痛点：
- **数据泄露 (Data Leakage)**：防止模型死记硬背特定人的特征。
- **类别不平衡 (Class Imbalance)**：确保基础类与异常类数据的合理配比。

---

## 2. 硬件配置与环境

### 2.1 设备参数

| 组件 | 型号/规格 | 备注 |
| :--- | :--- | :--- |
| **主控芯片** | ESP32-S3-N16R8 | 负责数据记录与预处理 |
| **惯性传感器** | MPU6050 (6轴) | 加速度计 + 陀螺仪 |
| **磁力计** | HMC5883L (3轴) | 辅助航向角计算 (可选) |

### 2.2 佩戴与采样设置

- **佩戴位置**：
    - 统一固定于 **腰部右侧（皮带处）** 或 **惯用手手腕**。
    - ⚠️ **注意**：全员必须统一佩戴位置，**严禁混用**不同位置的数据，否则会导致特征分布剧烈偏移。
- **采样率设置**：
    - 加速度/陀螺仪：**`50 Hz`** (满足奈奎斯特采样定理，且平衡存储压力)。
    - 磁力计：`10-20 Hz` (视 I2C 总线带宽而定)。

---

## 3. 被试方案 (Subject Protocol)

为避免模型只认识“某一个人”的动作（过拟合），必须严格遵循 **被试独立原则**。

### 3.1 人员要求
- **人数**：至少 **3名** 志愿者。
- **多样性**：建议涵盖不同体型（胖/瘦）、性别、年龄，以增加模型的泛化能力。

### 3.2 采集量标准
每位被试需完成以下任务：
- **覆盖度**：对 **每一类动作** 至少采集 **3次**。
- **时长**：每次持续时长 **≥ 60秒** (保证有足够的数据窗口进行切分)。

### 3.3 文件命名规范
采用结构化命名法，便于后续脚本自动解析：
> 格式：`Sub[ID]_Act[Action]_Trial[TrialNum].csv`

- 示例：`Sub01_Walk_Trial01.csv`
- 异常标记：若采集中途出现干扰，文件名后备注 `_bad` (如 `Sub01_Sit_Trial02_bad.csv`)，后期予以剔除。

---

## 4. 动作执行规范

| 类别 | 执行说明 | ⚠️ 关键注意事项 |
| :--- | :--- | :--- |
| **静坐 / 站立** | 保持自然姿态，不要刻意僵硬，允许微小生理晃动。 | 需记录开始和结束时的静止状态以便对齐时间戳。 |
| **平地步行 / 跑步** | 保持匀速，沿直线行走至少 20米往返。 | 避免急停急转；跑步时注意摆臂幅度的一致性。 |
| **上楼 / 下楼** | 正常步伐，手扶或不扶栏杆需全程保持一致。 | **高风险混淆项**：务必保证楼层高度一致，步频稳定。 |
| **跌倒 (Falling)** | **(注意安全!)** 在软垫上进行模拟跌倒（向前/向后/侧向）。 | **必须有保护人员在场！** 动作要干脆，触地后保持静止 2秒。 |

---

## 5. 标签与预处理规则 (关键！)

为了防止模型学习到错误的过渡特征（例如从坐到站的起立过程），必须执行 **过渡段裁剪**。

1. **打标方式**：
   采用“动作稳定后开始计时，动作结束前停止计时”的原则。
2. **过渡段切除 (Trimming)**：
   - 每个样本的 **前 2秒** 和 **后 2秒** 视为“动作切换期”（如从站起到迈步的过程）。
   - **处理操作**：在后续数据处理脚本中，必须将这前后各 2秒的数据 **丢弃**，只保留中间稳定的数据段。

---

## 6. 数据集划分原则 (防泄露铁律)

> 🛑 **绝对禁止**：将同一个人的数据拆分到训练集和测试集。这会导致测试集准确率虚高（99%+），但在实际应用中完全失效。

**正确做法 (LOSO / Subject-Independent)**：

- **训练集 (Train)**：被试 A + 被试 B 的所有数据。
- **验证集 (Val)**：从训练集中随机分出 20% 用于调参。
- **测试集 (Test)**：**被试 C** 的所有数据 (从未参与训练)。
