\# HAR-Project: 基于可穿戴传感器的人体活动识别系统



\## 1. 项目简介

本项目旨在构建一套完整的人体活动识别（HAR）系统，涵盖从底层硬件数据采集、传感器标定到上层模式识别算法的全链路闭环。

\## 3. 目录结构

\- `hardware/`: KiCad 原理图、PCB 布局及 BOM 表。

\- `data/`: 原始采集数据、公开数据集（UCI HAR）及预处理后的数据。

\- `src/`:

&#x20; - `acquire/`: ESP-IDF 固件及串口接收脚本。

&#x20; - `preprocess/`: 数据清洗、去重力、标定代码。

&#x20; - `features/`: 时域/频域特征提取与降维分析。

&#x20; - `models/`: 机器学习模型训练与 LOSO 评估脚本。

\- `docs/`: 项目计划书、采集协议及相关技术文档。
